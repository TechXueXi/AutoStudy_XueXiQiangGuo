****声明：仅供学习交流用，请24小时内删除****

1.需要配置模拟器，推荐雷电模拟器；如果自己有手机，也可以用Android手机，在default.ini中配置即可；

2.在使用过程中，请勿同时连接＞1个Android设备，请勿同时开启2个及以上模拟器；

3.推荐使用逍遥模拟器/雷电模拟器；
